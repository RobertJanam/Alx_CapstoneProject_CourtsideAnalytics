Welcome to Basketball Courtside Analytics

This is an API for tracking basketball games and player stats

Available API Endpoints

Authentication

  • POST/api/auth/register/ - Register new user
  • POST/api/auth/login/ - Login
  • GET/api/auth/me/ - View profile

Teams

  • GET/api/teams/ - List my teams
  • POST/api/teams/ - Create new team
  • POST/api/teams/join/ - Join team with code
  • GET/api/teams/{id}/ - View team details

Games

  • GET/api/teams/{id}/games/ - List team games
  • POST/api/teams/{id}/games/ - Create game (coach only)

You can use Postman to test the API endpoints
